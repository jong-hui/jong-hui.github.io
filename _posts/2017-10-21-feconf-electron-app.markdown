## Electron

웹 기술 기반의 크로스 플랫폼 데스크탑 앱 구축 기술

2013년 1월 시작

github에서 개발

### 구조

크로미움 + 노드 + 네이티브 API = 일렉트론

메인 + 렌더러 프로세스

### 무슨 이점?

프론트엔드 개발자 영여 확장

- 다양한 플랫폼

크로스 브라우징 이슈 해결

- 크로미움 기반 동작

### 버릴것은 버리자

기능 단순화

크로스브라우징이 큰 문제 - 일렉트론으로 감싸버림
