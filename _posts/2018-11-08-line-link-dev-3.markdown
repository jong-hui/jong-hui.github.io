### 엔터프라이즈 블록체인

기업환경에서의 블록체인

블록체인은 기업에 적용하기 사실 어려운 파트다.

기업은 서비스에 블록체인적용을 생각함

퓨어한 블록체인과는 척을 짐.

블로코는 비투비 비투씨 최고 숫자 기록

코인스택 : 인증, 투표, 거래, 전자문서 등

인증사업: 챌린지 리스판스 : 사용자 앱에 난스값보내고 서명

금융기관 송금: 은행 - 중앙기관에 다른은행에 돈보낸다고 알려줌 - 다른은행에서 확인 등등 어려움.

분산원장 :: 비가역성 :: 오너십
